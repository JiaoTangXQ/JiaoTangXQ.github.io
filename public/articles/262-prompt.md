# 第十三卷回收索引：前四卷里的 `prompt` 命令先重摆人模型系统工具的关系

## 实现链
`prompt` 不只是改一句话，它会先把人、模型、系统、工具的关系重新摆好，再让主循环、工具系统和恢复链继续往前走。

## 普通做法
更普通的做法，是把 prompt 当成一段普通用户文本，只负责补充内容。

## 为什么不用
Claude Code 不这么做，因为真正要改的不是一句回复，而是这段会话里谁能说话、谁能做事、谁来收口。

## 代价
代价是协作状态更复杂、主循环要求更高，但这也让长会话里的关系不会越跑越乱。
