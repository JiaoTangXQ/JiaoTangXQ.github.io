# 第十三卷回收索引：前四卷里的技能正文、工具许可、hooks、附件和恢复一起带进当前会话

## 实现链
`prompt` 命令会把技能正文、工具许可、hooks、附件和恢复一起带进当前会话，让后面的压缩、恢复和继续仍然认得这次工作到底是怎么开始的。

## 普通做法
更普通的做法，是只把一段 prompt 文本传进去，其他材料等后面需要时再补。

## 为什么不用
Claude Code 不这么做，因为后续的 resume、compact 和恢复判断都需要这些材料一开始就在会话里。

## 代价
代价是 prompt 载荷更重，session 状态也更丰富，但这正好把后续接续所需的上下文留住了。
